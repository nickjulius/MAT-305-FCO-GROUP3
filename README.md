# The Influence of Tiktok Affiliate Digital Marketing Strategies on the Purchasing Intention of Gen Z
**BSM AS 3B – Math Modeling Project**

---

## Project Overview
This project examines the influence of TikTok affiliate digital marketing strategies on the purchasing intention of Generation Z students.

The study applies quantitative analysis, descriptive statistics, and multiple linear regression modeling to determine how perceived usefulness, trust, promotion, consumer feedback, and social impact affect purchase intention.

---

### How to Run (MATLAB)

1. Open MATLAB
2. Set the working directory to the project folder
3. Run the main script:
 ```mathlab```
